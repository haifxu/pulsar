Solr sink connector

Solr sink 连接器将消息从 Pulsar 主题 取出，并将消息持久化到 Solr 索引库。

Configuration

Solr sink 连接器的配置具有以下属性。

属性

配置项类型Required默认值说明
solrUrlStringtrue" "（空字符串）
  • 在 SolrCloud 模式中使用逗号分隔的 zookeeper 主机（已使用 chroot）。
    Example
    localhost:2181,localhost:2182/chroot

  • 连接到单机模式下的 Solr 的 URL。
    Example
    localhost:8983/solr
    • solrModeStringtrueSolrCloud与 Solr 集群交互时的客户端模式。

    以下是可用的选项：
  • Standalone
  • SolrCloud
    • solrCollectionStringtrue" "（空字符串）Solr collection name to which records need to be written.
    solrCommitWithinMsintfalse10Solr 更新提交的时间（在百万秒内）。
    usernameStringfalse" "（空字符串）The username for basic authentication.

    Note: usename is case-sensitive.
    passwordStringfalse" "（空字符串）用于基本身份验证的密码。

    Note: password is case-sensitive.

    示例

    在使用 Solr sink 连接器之前，你需要通过以下方法之一创建一个配置文件。

    • JSON

      {
    "solrUrl": "localhost:2181,localhost:2182/chroot",
    "solrMode": "SolrCloud",
    "solrCollection": "techproducts",
    "solrCommitWithinMs": 100,
    "username": "fakeuser",
    "password": "fake@123"
}

    • YAML

      {
    solrUrl: "localhost:2181,localhost:2182/chroot"
    solrMode: "SolrCloud"
    solrCollection: "techproducts"
    solrCommitWithinMs: 100
    username: "fakeuser"
    password: "fake@123"
}
