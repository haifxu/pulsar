Cassandra sink connector

Cassandra sink 连接器将消息从 Pulsar 主题拖动到 Cassandra 集群。

Configuration

Cassandra sink 连接器的配置具有以下属性。

属性

配置项类型Required默认值说明
rootsStringtrue" "（空字符串）要连接的 Cassandra 主机列表，以逗号分割。
keyspaceStringtrue" "（空字符串）用来写入 Pulsar 消息的键空间（Keyspace）。

Note: keyspace should be created prior to a Cassandra sink.
keynameStringtrue" "（空字符串）Cassandra 列族（column family）的键名称。

The column is used for storing Pulsar message keys.

If a Pulsar message doesn't have any key associated, the message value is used as the key.
columnFamilyStringtrue" "（空字符串）The Cassandra column family name.

Note: columnFamily should be created prior to a Cassandra sink.
columnNameStringtrue" "（空字符串）The column name of the Cassandra column family.

The column is used for storing Pulsar message values.

示例

在使用 Cassandra sink 连接器之前，需要通过以下方式之一创建一个配置文件。

  • JSON

    {
    "roots": "localhost:9042",
    "keyspace": "pulsar_test_keyspace",
    "columnFamily": "pulsar_test_table",
    "keyname": "key",
    "columnName": "col"
}

  • YAML

    configs:
    roots: "localhost:9042"
    keyspace: "pulsar_test_keyspace"
    columnFamily: "pulsar_test_table"
    keyname: "key"
    columnName: "col"
